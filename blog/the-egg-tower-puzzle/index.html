The egg tower puzzle

6 Jun, 2020
13 min read (2,525 words)
  • math
  • julia

Here is a fun puzzle:

You are given two eggs, and access to a 100-storey tower. Both eggs are identical. The aim is to find out the highest floor from which an egg will not break when dropped out of a window from that floor. If an egg is dropped and does not break, it is undamaged and can be dropped again. However, once an egg is broken, that's it for that egg.

If an egg breaks when dropped from a floor, then it would also have broken from any floor above that. If an egg survives a fall, then it will survive any fall shorter than that.

The question is: What strategy should you adopt to minimize the number egg drops it takes to find the solution? (And what is the worst case for the number of drops it will take?)

Once you solved it for two eggs, can you solve it for three eggs? Can you find a generalized solution when you have a tower with xxx floors and if you have NNN eggs?

We will go through this problem for the 222 eggs and 333 eggs variant, and derive the solution mathematically. Then, we will generalize the problem with xxx floors and NNN eggs.

Two eggs#

We have 222 eggs to start with and a 100100100 storey tower to explore.

Let's consider what we would do if we had just one egg.

With just one egg, we could drop the egg from floor 111. If it breaks, we stop and can definitively say that for any floor 111 and above all eggs would break. If it doesn't break, we can proceed to floor 222 and repeat.

This means that for a tower with 100100100 floors we will need up to 100100100 drops to definitively say at which floor the egg would break. The above strategy gives us the minimum number of drops to guarantee that we find at which floor the egg would break. Or in other words, we can search NNN floors with NNN drops. Let's call the lowest floor in the tower at which a egg would break the breaking floor. Here, NNN is the breaking floor and N−1N-1N−1 is the highest floor from which an egg will not break. Once we can find the breaking floor, we can find the solution to this puzzle. The aim of this puzzle is to minimize the worst case when trying to guarantee finding the breaking floor.

What could we do differently with 222 eggs?

With 222 eggs, in fact we can explore the search space more efficiently. We can use the first egg to partition the total number of floors, and once the first egg breaks we can use the second egg to search floors within the last partition.

As an example, let's say you decide to drop the first egg from every 10th10^{th}10th floor. If it doesn't break at floor 101010, 202020, 303030, 404040 or 505050 but breaks at floor 606060 you can use the second egg to explore floors 515151 - 595959. If it finally breaks at floor 595959, you would have used 666 drops of the first egg and 999 drops of the second egg to find the breaking floor.

We can do even better though.

Let's assume that the minimum number of drops required to guarantee finding the breaking floor for a NNN storey tower is xxx. If when we drop the first egg and the egg breaks, we can use x−1x - 1x−1 drops of the second egg to find the breaking floor. We already know that we can search at most x−1x - 1x−1 floors with the 111 egg using x−1x - 1x−1 drops. So if our first drop was from a floor greater than xxx, we would not be able to guarantee finding the solution to this problem. This means that if we have 222 eggs we would want to drop the first egg from floor xxx, where xxx is also the minimum number of drops that will guarantee finding the breaking floor in a NNN storey tower.

When we drop the first egg from floor xxx, if the egg breaks, we can use the second egg to find which floor from 111 to x−1x - 1x−1 is the solution. If the egg doesn't break, now we have used 111 drop. Let's assume that the first egg breaks on the second drop. When this happens, we can use the remaining egg to explore x−2x - 2x−2 floors with x−2x - 2x−2 drops find the breaking floor. With x−2x - 2x−2 drops, we can search from floor x+1x + 1x+1 to floor x+(x−2)x + (x - 2)x+(x−2). This means that when we drop the first egg the second time, we should start from x+(x−2)+1x + (x - 2) + 1x+(x−2)+1, i.e., x+(x−1)x + (x - 1)x+(x−1), to allow for finding the breaking floor.

If the egg doesn't break on the second drop, we have now used 222 drops. If the egg is going to break on the third drop, we have to allow for searching x−3x - 3x−3 floors with the second egg. This means we should drop the first egg on our third attempt from floor number x+(x−1)+(x−3)+1x + (x - 1) + (x - 3) + 1x+(x−1)+(x−3)+1, i.e., x+(x−1)+(x−2)x + (x - 1) + (x - 2)x+(x−1)+(x−2). We can also say that with 333 drops and 222 eggs we are guaranteed to find the floor if it is within the first 3+(3−1)+(3−2)3 + (3 - 1) + (3 - 2)3+(3−1)+(3−2) floors, i.e. 666 floors.

Seeing the pattern here?

For a NNN storey tower, we need to ensure that with xxx drops we cover all the floors of the tower. That gives us this constraint.

x+(x−1)+(x−2)+(x−3)+…+1>=Nx + (x - 1) + (x - 2) + (x - 3) + \ldots + 1 >= Nx+(x−1)+(x−2)+(x−3)+…+1>=N

Which can be rewritten as:

∑k=1xk>=N\sum_{k=1}^{x} k >= N∑k=1x​k>=N

Or, in closed form as:

x×(1+x)2>=N\frac{x \times (1 + x)}{2} >= N2x×(1+x)​>=N

We know NNN is 100100100 in this problem.

x×(1+x)2>=100\frac{x \times (1 + x)}{2} >= 1002x×(1+x)​>=100

This tells us that with 141414 drops we can guarantee finding the breaking floor in a tower with up to 105105105 floors.

Three eggs#

We have already previously established the best strategy for 111 egg and 222 eggs.

Let's define some terminology:

f1(x)=xf_1(x) = xf1​(x)=x f2(x)=x×(1+x)2f_2(x) = \frac{x \times (1 + x)}{2}f2​(x)=2x×(1+x)​

where f1(x)f_1(x)f1​(x) and f2(x)f_2(x)f2​(x) are the number of floors that can be checked with 111 and 222 eggs respectively with xxx drops.

And we know that f2(x)f_2(x)f2​(x) can be written as:

f2(x)=∑k=1xkf_2(x) = \sum_{k=1}^{x} kf2​(x)=∑k=1x​k

So if we have 3 eggs and if the first egg breaks on the first drop, we want to ensure that we can check f2(x−1)f_2(x - 1)f2​(x−1) floors with the remaining x−1x - 1x−1 drops. That means we should drop the first egg from floor:

1+f2(x−1)1 + f_2(x - 1)1+f2​(x−1)

For the second drop, we can start from floor:

1+f2(x−1)+1+f2(x−2)1 + f_2(x - 1) + 1 + f_2(x - 2)1+f2​(x−1)+1+f2​(x−2)

For the third drop, we can start from floor:

1+f2(x−1)+1+f2(x−2)+1+f2(x−3)1 + f_2(x - 1) + 1 + f_2(x - 2) + 1 + f_2(x - 3)1+f2​(x−1)+1+f2​(x−2)+1+f2​(x−3)

Seeing a pattern emerge again?

Similar to before, the total number of floors we can check is constrained by NNN:

1+f2(x−1)+1+f2(x−2)+1+f2(x−3)+…+1>=N1 + f_2(x - 1) + 1 + f_2(x - 2) + 1 + f_2(x - 3) + \ldots + 1 >= N1+f2​(x−1)+1+f2​(x−2)+1+f2​(x−3)+…+1>=N

This can be rewritten as:

∑k=1x−1(1+f2(k))+1>=N\sum_{k=1}^{x-1} \left( 1 + f_2(k) \right) + 1 >= N∑k=1x−1​(1+f2​(k))+1>=N

1+∑j=1x−1(1+∑k=1jk)>=N1 + \sum_{j=1}^{x-1} \left( 1 + \sum_{k=1}^{j} k \right) >= N1+∑j=1x−1​(1+∑k=1j​k)>=N

x+∑j=1x−1∑k=1jk>=Nx + \sum_{j=1}^{x-1}\sum_{k=1}^{j} k >= Nx+∑j=1x−1​∑k=1j​k>=N

which results in:

x3+5x6>=N\frac{x^3 + 5x}{6} >= N6x3+5x​>=N

Solving this, we get 999 drops for 333 eggs. With just 999 drops, we can guarantee finding the breaking floor in a 129129129 storey tower.

NNN eggs#

Let's see if we can generalize this for NNN eggs.

With 111 egg and xxx drops, we know we can check xxx floors. Let's define this as f(x,1)f(x, 1)f(x,1), i.e.:

[f(x,1)=x(1)f(x, 1) = x \qquad(1)f(x,1)=x(1)]{#eq-f_x_1}

where f(x,n)f(x, n)f(x,n) is the floors that can be checked with xxx drops and nnn eggs.

We can safely say that with 000 drops or 000 eggs we can check 000 floors.

f(0,n)=f(x,0)=0f(0, n) = f(x, 0) = 0f(0,n)=f(x,0)=0

So we can write Equation 1{.quarto-xref} as the following:

f(x,1)=1+f(x−1,1)+f(x−1,0)f(x, 1) = 1 + f(x-1, 1) + f(x-1, 0)f(x,1)=1+f(x−1,1)+f(x−1,0)

For 222 eggs and xxx drops, the number of floors we can check, i.e. f(x,2)f(x, 2)f(x,2), is:

[f(x,2)=∑k=1xk(2)f(x, 2) = \sum_{k=1}^{x} k \qquad(2)f(x,2)=∑k=1x​k(2)]{#eq-f_x_2}

We can expand this as the following:

f(x,2)=1+∑k=1x−1k+x−1f(x, 2) = 1 + \sum_{k=1}^{x-1} k + x - 1f(x,2)=1+∑k=1x−1​k+x−1

and substitute in Equation 1{.quarto-xref} to get:

f(x,2)=1+f(x−1,2)+f(x−1,1)f(x, 2) = 1 + f(x-1, 2) + f(x-1, 1)f(x,2)=1+f(x−1,2)+f(x−1,1)

And, for 333 eggs and xxx drops, the number of floors we can check, i.e. f(x,2)f(x, 2)f(x,2), is:

f(x,3)=x+∑j=1x−1∑k=1jkf(x, 3) = x + \sum_{j=1}^{x-1}\sum_{k=1}^{j} kf(x,3)=x+∑j=1x−1​∑k=1j​k

Again, we can expand it and rearrange some terms:

f(x,3)=x+∑j=1x−2∑k=1jk+∑k=1x−1kf(x, 3) = x + \sum_{j=1}^{x-2}\sum_{k=1}^{j} k + \sum_{k=1}^{x-1} kf(x,3)=x+∑j=1x−2​∑k=1j​k+∑k=1x−1​k

f(x,3)=1+x−1+∑j=1x−2∑k=1jk+∑k=1x−1kf(x, 3) = 1 + x - 1 + \sum_{j=1}^{x-2}\sum_{k=1}^{j} k + \sum_{k=1}^{x-1} kf(x,3)=1+x−1+∑j=1x−2​∑k=1j​k+∑k=1x−1​k

and substitute in Equation 2{.quarto-xref} to get:

f(x,3)=1+f(x−1,3)+f(x−1,2)f(x, 3) = 1 + f(x - 1, 3) + f(x - 1, 2)f(x,3)=1+f(x−1,3)+f(x−1,2)

We can see a pattern emerging here. The generalized equation can be written like so:

[f(x,n)=1+f(x−1,n)+f(x−1,n−1)(3)f(x, n) = 1 + f(x - 1, n) + f(x - 1, n - 1) \qquad(3)f(x,n)=1+f(x−1,n)+f(x−1,n−1)(3)]{#eq-f_x_n}

This is also known as a recurrence relation.

This result can be reasoned through intuition as well. To find the total floors you can check with nnn eggs and xxx drops, it will be 111 (your first drop) plus the maximum number of floors you can check with nnn eggs and x−1x-1x−1 eggs (if the first egg does not break) plus the maximum number of floors you can check with n−1n-1n−1 eggs with x−1x-1x−1 eggs (if the first egg does break).

If we wanted to, we could expand this recurrence relation1.

Here is the expansion increasing the depth in the direction of number of drops:

f(x,n)=1+f(x−1,n)+f(x−1,n−1)f(x, n) = 1 + f(x - 1, n) + f(x - 1, n - 1)f(x,n)=1+f(x−1,n)+f(x−1,n−1)

f(x,n)=3+f(x−2,n)+2f(x−2,n−1)+f(x−2,n−2)f(x, n) = 3 + f(x - 2, n) + 2f(x - 2, n - 1) + f(x - 2, n - 2)f(x,n)=3+f(x−2,n)+2f(x−2,n−1)+f(x−2,n−2)

f(x,n)=7+f(x−3,n)+3f(x−3,n−1)+3f(x−3,n−2)+f(x−3,n−3)f(x, n) = 7 + f(x - 3, n) + 3f(x - 3, n - 1) + 3f(x - 3, n - 2) + f(x - 3, n - 3)f(x,n)=7+f(x−3,n)+3f(x−3,n−1)+3f(x−3,n−2)+f(x−3,n−3)

f(x,n)=15+f(x−4,n)+4f(x−4,n−1)+6f(x−4,n−2)+4f(x−4,n−3)+f(x−4,n−4)f(x, n) = 15 + f(x - 4, n) + 4f(x - 4, n - 1) + 6f(x - 4, n - 2) + 4f(x - 4, n - 3) + f(x - 4, n - 4)f(x,n)=15+f(x−4,n)+4f(x−4,n−1)+6f(x−4,n−2)+4f(x−4,n−3)+f(x−4,n−4)

f(x,n)=31+f(x−5,n)+5f(x−5,n−1)+10f(x−5,n−2)+10f(x−5,n−3)+5f(x−5,n−4)+f(x−5,n−5)f(x, n) = 31 + f(x - 5, n) + 5f(x - 5, n - 1) + 10f(x - 5, n - 2) + 10f(x - 5, n - 3) + 5f(x - 5, n - 4) + f(x - 5, n - 5)f(x,n)=31+f(x−5,n)+5f(x−5,n−1)+10f(x−5,n−2)+10f(x−5,n−3)+5f(x−5,n−4)+f(x−5,n−5)

f(x,n)=63+f(x−6,n)+6f(x−6,n−1)+15f(x−6,n−2)+20f(x−6,n−3)+15f(x−6,n−4)+6f(x−6,n−5)+f(x−6,n−6)f(x, n) = 63 + f(x - 6, n) + 6f(x - 6, n - 1) + 15f(x - 6, n - 2) + 20f(x - 6, n - 3) + 15f(x - 6, n - 4) + 6f(x - 6, n - 5) + f(x - 6, n - 6)f(x,n)=63+f(x−6,n)+6f(x−6,n−1)+15f(x−6,n−2)+20f(x−6,n−3)+15f(x−6,n−4)+6f(x−6,n−5)+f(x−6,n−6)

Eagle eye readers will notice a pattern.

It turns out that this expansion is related to the binomial coefficients.

If we had infinite number of eggs, you'd see that the first element is the only term that contributes to f(x,n)f(x, n)f(x,n), since the others will be 000 for x−dx - dx−d where ddd is the depth in the triangle. That is to say, if we had infinite eggs, with 6 drops we can guarantee checking 63 floors.

Using this, we can say that, with xxx drops we can guarantee checking 2x−12^x - 12x−1 floors if we had infinite eggs2.

You'll notice that the upper right corner of the table stays the same if you increase the number of eggs you have at your disposal. You can see this even more clearly in this visualization.

There's a minimal number of drops required to guarantee that you will find the breaking floor, even if you have unlimited eggs.

If you want to check 777 floors, as long as you have more than 333 eggs, you can have to use a minimum of 3 drops to guarantee finding the breaking floor.

For 100 floors, it is 777 drops. If you had unlimited eggs, you would use the largest partition possible. That means dividing the total floors by 222.

Partitioning the floors equally or in other words bisecting the floors, and exploring the partition of interest using the same strategy is the most efficient way of finding the breaking floor.

So there you go! We have solved the general case for the egg and tower puzzle.
